# About Section Hero Image

Place your lab's hero image here as `lab_hero.jpg` (or any image format).

Recommended specifications:
- **Format**: JPG, PNG, or WebP
- **Dimensions**: 1920x1080 pixels (16:9 aspect ratio) or similar
- **Size**: Under 2MB for optimal loading speed
- **Content**: Lab group photo, research facility, or representative image

The image will be displayed as a large hero banner on the About page with overlay text.

## How to add your image:

1. Copy your image file to this directory
2. Rename it to `lab_hero.jpg` (or update the path in `_data/about.yml`)
3. The image will automatically appear on the About page

## Alternative formats:

If using a different format or name, update the `hero_image` field in `_data/about.yml`:

```yaml
hero_image: "/assets/images/about/your_image_name.png"
```
